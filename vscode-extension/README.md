Provides TextMate-based syntax highlighting for Shiden `.sd` files. This extension only contributes the grammar and language configuration so VS Code can colorize `.sd` files; it does not provide formatting, diagnostics, completions, or snippets.

Features
- Syntax highlighting for `.sd` files (scopes defined in `syntaxes/shiden.tmLanguage.json`)

How to build
1. cd vscode-extension
2. vsce package
